# Как запустить dev-сервер

## Способ 1: Через терминал в VS Code / Cursor

1. Откройте терминал в редакторе:
   - **VS Code/Cursor**: `View` → `Terminal` или нажмите `` Ctrl+` `` (обратная кавычка)
   - Или: `Terminal` → `New Terminal`

2. В терминале выполните:

   ```bash
   npm run dev
   ```

3. Вы увидите вывод:

   ```text
   ▲ Next.js 14.x.x
   - Local:        http://localhost:3000
   - Ready in X ms
   ```

4. Откройте браузер и перейдите на `http://localhost:3000`

## Способ 2: Через системный терминал (Terminal.app на Mac)

1. Откройте приложение **Terminal** (находится в `/Applications/Utilities/`)

2. Перейдите в директорию проекта:

   ```bash
   cd "/Users/olgapuchkova/Documents/Документы — MacBook Air — Olga/PERSONAL/МОИ ПРОЕКТЫ/ПЛАТФОРМА ЖЕНСКОГО ЗДОРОВЬЯ/Проект Без | паузы/Бот Без|паузы/САЙТ"
   ```

3. Запустите dev-сервер:

   ```bash
   npm run dev
   ```

## Как остановить сервер

В терминале, где запущен сервер, нажмите:

- `Ctrl + C` (на Mac и Linux)
- Или закройте окно терминала

## Как найти запущенный сервер

Если сервер уже запущен, но вы не видите терминал:

1. **В VS Code/Cursor**: Проверьте все открытые панели терминала
   - Нажмите `` Ctrl+` `` несколько раз
   - Или `Terminal` → `Show All Terminals`

2. **В системном терминале**: Откройте новое окно терминала и выполните:

   ```bash
   lsof -ti:3000
   ```

   Это покажет PID процесса. Затем:

   ```bash
   kill -9 <PID>
   ```

   (замените `<PID>` на номер процесса)

3. **Перезапустите сервер** после остановки старого процесса

## Полезные команды

```bash
# Проверить, запущен ли сервер на порту 3000
lsof -ti:3000

# Остановить все процессы на порту 3000
kill -9 $(lsof -ti:3000)

# Запустить dev-сервер
npm run dev

# Запустить в фоновом режиме (не рекомендуется для разработки)
npm run dev &
```

## Если порт 3000 занят

Если вы видите ошибку "Port 3000 is already in use":

1. Остановите процесс:

   ```bash
   kill -9 $(lsof -ti:3000)
   ```

2. Или используйте другой порт:

   ```bash
   PORT=3001 npm run dev
   ```
