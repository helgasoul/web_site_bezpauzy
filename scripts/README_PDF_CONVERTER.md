# Конвертер PDF в Markdown

Скрипт для конвертации PDF файлов в формат Markdown.

## Установка зависимостей

```bash
npm install pdf-parse
```

## Использование

### Конвертировать один файл

```bash
npm run pdf-to-md "litrature/Загруженные в Без|паузы/HRT – Practical prescribing.pdf" "litrature/zgt_text"
```

Или напрямую:

```bash
node scripts/pdf-to-markdown.js "путь/к/файлу.pdf" "выходная/папка"
```

### Конвертировать все PDF в папке

```bash
npm run pdf-to-md "litrature/Загруженные в Без|паузы/" "litrature/zgt_text" --all
```

## Примеры

```bash
# Конвертировать один файл о ЗГТ
npm run pdf-to-md "litrature/Загруженные в Без|паузы/HRT – Practical prescribing.pdf" "litrature/zgt_text"

# Конвертировать все PDF о ЗГТ
npm run pdf-to-md "litrature/Загруженные в Без|паузы/" "litrature/zgt_text" --all
```

## Что делает скрипт

1. Читает PDF файл
2. Извлекает текст и метаданные
3. Форматирует текст в Markdown:
   - Определяет заголовки и подзаголовки
   - Форматирует списки
   - Разбивает длинные абзацы
4. Сохраняет результат в указанную папку

## Формат выходного файла

- Имя файла: `имя-pdf-файла.md`
- Содержит:
  - Заголовок с названием
  - Метаданные (автор, дата создания)
  - Отформатированный текст

## Примечания

- Скрипт пытается автоматически определить структуру документа
- Для лучших результатов может потребоваться ручная правка
- Некоторые PDF могут содержать изображения, которые не будут извлечены
