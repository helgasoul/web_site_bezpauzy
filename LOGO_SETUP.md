# Инструкция по добавлению логотипа

## Проблема

Логотип не отображается, потому что файл нужно переместить в папку `public/`.

## Решение

### Шаг 1: Переместить файл логотипа

**Через Finder (рекомендуется):**

1. Откройте Finder
2. Перейдите в папку проекта (где находится `package.json`)
3. Найдите файл: `ChatGPT Image Jul 16, 2025 at 07_36_47 PM.png`
4. Переименуйте его в `logo.png` (для удобства)
5. Переместите в папку `public/` (создайте её, если нет)

**Или через терминал Cursor:**

В терминале Cursor (где вы запускали `npm run dev`) выполните:

```bash
# Создать папку public (если нет)
mkdir -p public

# Переместить и переименовать файл
mv "ChatGPT Image Jul 16, 2025 at 07_36_47 PM.png" public/logo.png
```

### Шаг 2: Проверка

После перемещения файла:

1. Убедитесь, что файл находится в `public/logo.png`
2. Next.js автоматически перезагрузится
3. Логотип должен появиться в Header

### Шаг 3: Если нужно изменить размер

Если размер логотипа нужно изменить, откройте `components/layout/Header.tsx` и измените параметры:

```tsx
<Image
  src="/logo.png"
  alt="Без |Паузы"
  width={48}  // измените это значение
  height={48} // измените это значение
  className="object-contain h-12 w-auto"
  priority
/>
```

## Текущая настройка

- Размер: 48x48px
- Позиция: слева от текста "Без |Паузы"
- Отступ: space-x-3 (12px между логотипом и текстом)

## Альтернатива: использовать оригинальное имя файла

Если хотите использовать оригинальное имя файла, можно изменить путь в Header.tsx:

```tsx
src="/ChatGPT Image Jul 16, 2025 at 07_36_47 PM.png"
```

Но лучше переименовать файл в `logo.png` для удобства.
