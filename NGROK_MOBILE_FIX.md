# Исправление проблемы с ngrok на мобильных устройствах

## Проблема
Ngrok free версия показывает предупреждающую страницу на мобильных устройствах, которая блокирует доступ к сайту.

## Решение

### Вариант 1: Использовать заголовок для обхода предупреждения

При открытии сайта на мобильном устройстве через ngrok, добавьте заголовок `ngrok-skip-browser-warning: true` в запросы.

**Для тестирования:**
1. Откройте сайт на телефоне
2. Если видите предупреждающую страницу ngrok, нажмите "Visit Site" или "Continue"
3. После этого сайт должен работать

### Вариант 2: Использовать ngrok с авторизацией (рекомендуется)

Если у вас есть аккаунт ngrok:
```bash
# Остановите текущий ngrok
# Затем запустите с авторизацией:
ngrok http 3000 --authtoken YOUR_AUTH_TOKEN
```

### Вариант 3: Использовать другой туннель

Альтернативы ngrok:
- **Cloudflare Tunnel** (бесплатно, без предупреждений)
- **localtunnel** (бесплатно, но менее стабильно)
- **serveo** (бесплатно, SSH-based)

✅ Переменные окружения настроены для development режима

## Проверка

После перезапуска dev-сервера:
1. Убедитесь, что ngrok запущен: `ps aux | grep ngrok`
2. Проверьте URL: `curl -s http://localhost:4040/api/tunnels | grep public_url`
3. Откройте URL на телефоне
4. Если видите предупреждение ngrok, нажмите "Visit Site"

## Важно

⚠️ **Ngrok free версия всегда показывает предупреждающую страницу на мобильных устройствах** - это ограничение бесплатного плана. Для production используйте платный план ngrok или другой туннель.
