# Инструкция по добавлению изображения в Hero секцию

## Что нужно сделать

Изображение `portret-zensiny-na-den-zensiny.jpg` нужно переместить в папку `public/` и переименовать.

### Через терминал Cursor (рекомендуется)

В терминале Cursor выполните:

```bash
# Переместить и переименовать файл
mv "portret-zensiny-na-den-zensiny.jpg" public/hero-women.jpg
```

### Через Finder

1. Откройте Finder
2. Найдите файл `portret-zensiny-na-den-zensiny.jpg` в корне проекта
3. Переименуйте его в `hero-women.jpg`
4. Переместите в папку `public/`
